My Portfolio

Here is some of my work that I've done in various programming languages.

Project Source

Journalling application that allows users to journal traditionally (type and write out their day) as well as with their voice. This application targets users who are too busy to consistently make time for journalling and therefore, we aimed to eliminate resistance so that a user walking back home could journal on the way back without sitting down. We utilized generative AI so that once you record your journal, you have the option of getting feedback and suggestions. Using AI, we provide users two different forms of recorded journalling, a freeform where users journal however they want and a guided where users are prompted questions to help them voice their thoughts. Utilized Python, HTML/CSS, and Javascript for back end prototype and Figma for front end prototype.
